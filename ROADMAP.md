# Roadmap

This repository contains reusable GitHub Actions templates and related project scaffolding.

## Near-term goals

- Improve template clarity and consistency.
- Add more examples for common CI/CD use cases.
- Expand documentation for customization and reuse.
- Make templates easier to copy into new repositories.

## Planned areas of work

### Workflow templates
- Build and test workflows.
- Linting and formatting workflows.
- Release and publishing workflows.
- Security and dependency scanning workflows.

### Documentation
- Add usage examples for each template.
- Document required inputs and expected outputs.
- Explain common customization patterns.
- Provide troubleshooting notes for frequent issues.

### Quality improvements
- Simplify variable and secret naming.
- Reduce duplication across templates.
- Improve compatibility with common repository layouts.
- Add validation examples where helpful.

## Contribution priorities

Contributions are most helpful when they:
- Fix real problems.
- Improve reuse across many projects.
- Add documentation or examples.
- Keep templates simple and maintainable.

## Roadmap notes

This roadmap is a living document and may change based on user feedback, maintainer capacity, and project needs.
