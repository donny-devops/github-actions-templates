# Support

Thank you for using `github-actions-templates`.

## Getting help

If you need help using a template, please:
- Review the repository documentation first.
- Check existing issues and discussions for similar questions.
- Open a new issue if your question is not already answered.

## What support covers

This repository supports:
- Questions about template usage.
- Bug reports for broken workflows or examples.
- Requests for clarification in documentation.
- Improvements that make the templates easier to reuse.

## What support does not cover

This repository does not provide:
- General GitHub support.
- Support for custom infrastructure or third-party services.
- Debugging for unrelated repository or organization settings.
- Help with private environments we cannot access.

## How to report a problem

When opening an issue, include:
- The template or workflow file involved.
- A short description of the problem.
- The expected behavior.
- The actual behavior.
- Any logs, screenshots, or relevant configuration.

## Response expectations

Maintainers will try to respond as time allows. Response time may vary depending on the issue volume and project activity.

## Security issues

Do not post sensitive security details in a public issue. Use the project’s private security contact process if one is available.
